# GENERATION SETTINGS RECOMMENDATIONS

## For Stable Diffusion / FLUX:
```
Recommended settings:
- Steps: 30-50
- CFG Scale: 7-9
- Sampler: DPM++ 2M Karras or Euler a
- Size: 1024x1024 (scale down for mobile)
- Negative prompt: "scary, horror, blood, violence, realistic photo, photograph, ugly, deformed, blurry, low quality, text, watermark, signature"
```

## For Character/Creature Models:
```
Additional positive keywords: "front view, character sheet, game sprite, clean silhouette"
Additional negative: "multiple views, turnaround, complex background"
```

## For UI Elements:
```
Additional positive keywords: "vector art, flat design, icon, symbolic, minimal"
Additional negative: "3D render, realistic, complex shading"
```

# POST-PROCESSING NOTES

After generation:
1. Remove backgrounds (use remove.bg or similar)
2. Resize for mobile (512x512 or 256x256 for sprites)
3. Create sprite atlases for animations
4. Add Unity-compatible transparency
5. Optimize file size for mobile

For animated intruders:
- Generate 3-4 pose variations
- Create sprite sheets
- Or use as reference for 3D modeling
